#Icons

This is a list of all icons available sorted in categories. Its a list where primary and secondary icons are combined. You may be interested to read more about what the difference is and how to import.

  • for Web there are both icons as SVG and ready to use React Components.
  • for iOS there is a custom PDF package and a categorized PDF package.
  • for Android there is a custom XML (drawable) package and a categorized XML (drawable) package.
