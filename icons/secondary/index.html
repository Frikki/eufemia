#Secondary Icons

The Secondary Icons are an addition to the Primary Icons. They extend the possibility to have more, not mainly used icons. They get not shipped integrated as the Primary Icons do.

The Secondary Icons can be extended infinitely.

Technically: The consequence is that they have to be imported where ever they have to be used.

#React example usage

import { hamburger_medium as HamburgerIcon, bubble } from 'dnb-ui-lib/icons'
<HamburgerIcon />/* <-- is not recommended, but possible */
<Icon icon={HamburgerIcon} />
<Button icon={bubble} />
