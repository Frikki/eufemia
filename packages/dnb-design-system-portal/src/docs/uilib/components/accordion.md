## Accordion with a single container

A single container is used only for wider screens (desktop).

<ComponentBox>
{`
<Accordion.Group
	variant="outlined"
	// prerender
	prevent_rerender
	single_container
	remember_state
	// allow_close_all
>

		// expanded={false}
		bottom
		id="remembered_state-1"
		title="Title1"
		description="Description1"
		// element="h2"
		// heading
		// heading={Heading}
		// heading_level="3"

			Description 3

			</P>
		</Accordion.Content>
    </Accordion>
</Accordion.Group>
`}
</ComponentBox>
