## ErrorCard Properties

| Properties                                  | Description                                                                            |
| ------------------------------------------- | -------------------------------------------------------------------------------------- |
| `title`                                     | _(required)_ Title text of the component (ReactNode)                                   |
| `message`                                   | _(required)_ Message/description text (ReactNode)                                      |
| `onTryAgainClick`                           | _(required)_ Function is called when the button is clicked                             |
| `className`                                 | _(optional)_ Option to overwrite stying of the outer wrapper                           |
| `variant`                                   | _(optional)_ Styling property, "slim" "default". defaults to default                   |
| `isCentered`                                | _(optional)_ Boolean that enables a centered version of the component                  |
| `customActions`                             | _(optional)_ Accepts any react element and will render them to the right of the button |
| [Space](/uilib/components/space/properties) | _(optional)_ spacing properties like `top` or `bottom` are supported.                  |
